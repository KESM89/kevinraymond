This is a sample post.

When you want to publish a new article, add a new file inside `_posts` using this naming format:

`YYYY-MM-DD-title-goes-here.md`

Example:

`2026-04-01-how-to-order-a-transition.md`

At the top of each post, use front matter like this:

```yaml
---
title: Your Post Title
description: One sentence that explains what the post is about.
categories: [sheet-metal]
---
```

Then write the post in normal Markdown below it.
